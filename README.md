Official website and landing page for DeclareUI — hosted on GitHub Pages.

## About

This is the source for [declare-ui.github.io](https://declare-ui.github.io), the public-facing landing page that introduces DeclareUI, links to documentation, and showcases the ecosystem.

## Development

```bash
pnpm install
pnpm dev
```
