See it in action
One source. Native output.
A single YAML definition compiles to idiomatic code for each framework.
button.ui.yaml source 
component: Button

props:
  variant:
    type: enum
    values: [primary, secondary, ghost]
    default: primary
  label:
    type: string
    required: true

template:
  tag: button
  classes:
    base: "px-4 py-2 rounded-lg font-medium"
    variants:
      variant:
        primary: "bg-blue-600 text-white"
        secondary: "bg-gray-100 text-gray-900"
  children:
    - tag: span
      content: "$props.label"
Button.tsx react output 
import { forwardRef } from 'react';
import { cn } from '../utils/cn';

export interface ButtonProps {
  variant?: 'primary' | 'secondary' | 'ghost';
  label: string;
}

export const Button = forwardRef<
  HTMLButtonElement, ButtonProps
>(({ variant = 'primary', label }, ref) => {
  const classes = cn(
    'px-4 py-2 rounded-lg font-medium',
    {
      primary: 'bg-blue-600 text-white',
      secondary: 'bg-gray-100 text-gray-900',
    }[variant]
  );

  return (
    <button ref={ref} className={classes}>
      <span>{label}</span>
    </button>
  );
});
Features
Everything you need to ship UI everywhere.
Built for teams that refuse to choose between consistency and developer experience.
{}

Declarative-First

YAML/JSON as the source of truth. No JSX, no framework-specific syntax. Readable by designers, PMs, and backend devs.

T

Tailwind Native

Classes resolved at build-time for every target. Shadow DOM support via Constructable Stylesheets. Zero runtime overhead.

</>

Native Bindings

Output is idiomatic for each framework — real React hooks, Vue Composition API, Svelte runes. Not wrappers around Web Components.

TS

End-to-End Types

Props defined in YAML generate TypeScript interfaces for every target. Full autocomplete, full safety.

✓

Declarative Tests

Write tests in YAML alongside components. Compiled to vitest + testing-library and executed against all targets automatically.

⚙

Plugin System

Custom generators for any target — Flutter, SwiftUI, Kotlin Compose. The AST is your API.
